# Nails De Maison — Website

The official website for **Nails De Maison**, a nail salon in Buford, GA.

Live booking is powered by **Square Appointments** (embedded in the "Book Now" popup).

## What's here
- `index.html` — the site (open this)
- `site.css`, `colors_and_type.css` — styles
- `*.jsx` — page sections (React, compiled in-browser)
- `assets/` — logo, photos, background textures

## Run locally
Just open `index.html` in a browser. (For the Square booking calendar to load,
the site must be served over **https** — it works once deployed, see below.)

## Deploy
This is a static site — it can be hosted free on **GitHub Pages**, **Netlify**, or
**Cloudflare Pages**. See deployment notes shared separately.
